# 지시사항

아래 GitHub 레포지토리들에 대한 HN(Hacker News)과 Reddit 커뮤니티 반응을 분석해주세요.

## 요구사항
- sentiment는 커뮤니티 반응의 긍정/중립/부정 비율을 정수 퍼센트로 산출합니다 (합계 100)
- insights는 커뮤니티 반응에서 주목할 만한 내용을 최대 3개, 한국어로 작성합니다
- 각 insight에는 해당 내용의 tone을 positive, neutral, negative 중 하나로 지정합니다
- 기술 용어는 영어 그대로 사용합니다 (예: "React 기반", "Rust 구현")
- 각 결과에는 원본 레포지토리의 ID(full_name)를 반드시 포함해야 합니다
